# rickbot
Plays srt subtitles over specified ros2 topic.

Default topic is /chatter

## Use
Clone, build and run with an srt file of your choice
```
ros2 run rickbot rickbotmain ~/Downloads/Rick-Astley-Never-Gonna-Give-You-Up.srt
```
